# 밥사주세요 앱 Firebase 프로젝트
새내기 및 대학생들을 위한 맛집 프로젝트

## 핵심기능
- 근처 식당, 맛집
- 식당에 갈 시간 걸어두기
- 리뷰
- 찜(즐겨찾기)

## 일반 기능
- 회원
    - 이메일 로그인, 이메일 회원가입
    - 로그아웃
    - 소셜 로그인, 소셜 회원가입
    - 비밀번호 찾기
- 푸시
    - 같은 시간대 선택한 사람 있으면 푸시
- 설정
    - 공지사항
    - 푸시 On / Off
